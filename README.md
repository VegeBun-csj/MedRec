# MedRec
This is a hospital management based on the hyperledger fabric and IPFS

- the chaincode is coded by golang language
- the SDK is using Node.js SDK
- js-IPFS

### prerequest:

node v10.19 and go >=1.13



### start up Network

only start the start.sh shell

```shell
./start.sh up
```



### start the App

```
npm install
```
